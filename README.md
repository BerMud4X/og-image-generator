# OG Image Generator

Generate dynamic Open Graph images on the fly with custom templates. Live editor, copy URL, download PNG.

**[→ Live demo](https://og-image-generator-ruddy-three.vercel.app/)**

![OG Image Generator preview](https://og-image-generator-ruddy-three.vercel.app/api/og?title=OG+Image+Generator&subtitle=Generate+dynamic+OG+images+live&template=terminal)

> _The image above is generated by this tool itself — meta._

---

## Why

Every SaaS, blog, and landing page needs Open Graph images for social sharing. Designing them manually for every page sucks. This tool lets you generate them on demand from URL parameters — no design tool needed, no image hosting needed.

## Features

- **4 distinct templates** — Minimal Dark, Gradient Bold, Terminal, Magazine
- **Live editor** — type, see the preview update in 300ms
- **Copy URL** — paste anywhere as your `<meta property="og:image">`
- **Download PNG** — get the file directly
- **HTML snippet** ready to paste into your `<head>`
- **Edge-rendered** — fast, cached by Vercel's CDN
- **Custom Google Fonts** loaded per template (Inter, JetBrains Mono, Playfair Display)

## Usage

### As an HTML meta tag

Copy this into your page `<head>`:

```html
<meta
  property="og:image"
  content="https://og-image-generator-ruddy-three.vercel.app/api/og?title=Your+Title&subtitle=Your+subtitle&template=gradient-bold"
/>
```

### Direct URL

```
https://og-image-generator-ruddy-three.vercel.app/api/og?title=...&subtitle=...&template=...
```

### Query parameters

| Param | Type | Required | Default | Description |
|---|---|---|---|---|
| `title` | string | no | `Hello World` | Main heading |
| `subtitle` | string | no | `Built with @vercel/og` | Secondary text |
| `template` | enum | no | `minimal-dark` | One of: `minimal-dark`, `gradient-bold`, `terminal`, `magazine` |

## Tech stack

- [Next.js 16](https://nextjs.org/) — App Router, edge runtime
- [@vercel/og](https://vercel.com/docs/functions/og-image-generation) (via `next/og`) — image generation
- [Satori](https://github.com/vercel/satori) — JSX → SVG → PNG
- [Tailwind CSS v4](https://tailwindcss.com/) — styling
- [shadcn/ui](https://ui.shadcn.com/) — UI components
- [Vercel](https://vercel.com/) — hosting

## How it works

1. The browser hits `/api/og?title=...&subtitle=...&template=...`
2. The route handler picks the template from a registry based on the `template` param
3. Each template declares the Google Fonts it needs; they're fetched in parallel from Google's CSS API
4. Satori renders the template's JSX to SVG, then to PNG
5. PNG is returned and cached by Vercel's CDN

## Local development

```bash
git clone https://github.com/BerMud4X/og-image-generator.git
cd og-image-generator
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Adding a new template

1. Create `src/app/api/og/templates/<name>.tsx` — exports `default` (the JSX function) and `fonts` (the Google Fonts array)
2. Register it in `src/app/api/og/templates/index.ts`
3. Add a `<SelectItem>` in `src/app/page.tsx`
